## Technical Details

### Adaptive Icon Structure

```
Android 8.0+ devices use two-layer adaptive icons:
- Background layer: Solid #2563EB blue
- Foreground layer: Icon with transparency
- Result: Works with any launcher shape (circle, square, rounded, teardrop)
```

### Splash Screen Sequence

```
1. Native Android splash (instant)
   - Blue gradient background
   - No icon, just color
   
2. Flutter splash screen (1.8s)
   - Animated typography
   - Smooth transitions
   - Professional appearance
   
3. Main app (fade transition)
   - Seamless experience
```

### Android 12+ Splash Screen API

```xml

<item name="android:windowSplashScreenBackground">#2563EB</item><item
name="android:windowSplashScreenAnimatedIcon">@android:color/transparent
</item>
```

This ensures no icon appears during the native splash on modern Android devices.
