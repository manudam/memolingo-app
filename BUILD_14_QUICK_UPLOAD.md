## 🆘 Troubleshooting

### Upload Fails

- Check internet connection
- Verify IPA file isn't corrupted
- Try command line upload
- Contact Apple Support

### Processing Takes Too Long (>2 hours)

- Normal: 15-30 minutes
- Check App Store Connect status
- Email should arrive when complete
- Contact Apple if stuck

### Can't Select Build

- Wait for processing to complete
- Refresh App Store Connect page
- Check build is for iOS (not simulator)
- Verify version numbers match
