# MemoLingo

A modern, gamified language learning app built with Flutter.

## Features

- Multi-language support with 9 languages
- Interactive practice sessions with gamified UI
- Detailed progress tracking, XP, and streaks
- Spaced Repetition System for efficient learning
- In-app purchases for category packs
- Cross-platform support (iOS, Android, iPad)
